# Reinforcement Learning Resources

## Beginner Resources

### Courses

- [Introduction to Reinforcement Learning with David Silver](https://www.youtube.com/playlist?list=PLqYmG7hTraZDM-OYHWgPebj2MfCFzFObQ) - DeepMind's RL course by David Silver
- [Reinforcement Learning Explained](https://www.edx.org/course/reinforcement-learning-explained) - Microsoft's introduction to RL
- [Practical RL](https://github.com/yandexdataschool/Practical_RL) - Yandex's practical course on RL

### Books

- [Reinforcement Learning: An Introduction](http://incompleteideas.net/book/the-book-2nd.html) - The classic RL textbook by Sutton and Barto
- [Grokking Deep Reinforcement Learning](https://www.manning.com/books/grokking-deep-reinforcement-learning) - Beginner-friendly introduction to RL
- [Deep Reinforcement Learning Hands-On](https://www.packtpub.com/product/deep-reinforcement-learning-hands-on-second-edition/9781838826994) - Practical guide to RL implementation

### Tutorials

- [Spinning Up in Deep RL](https://spinningup.openai.com/en/latest/) - OpenAI's educational resource on RL
- [Stable Baselines3 Documentation](https://stable-baselines3.readthedocs.io/en/master/) - Tutorials for the RL library
- [RL Tutorial Series](https://www.youtube.com/playlist?list=PL-9x0_FO_lgntlBuxS42lXy2aSXiCGEar) - Practical RL tutorials

## Intermediate Resources

### Courses

- [Reinforcement Learning Specialization](https://www.coursera.org/specializations/reinforcement-learning) - University of Alberta's RL specialization
- [Deep Reinforcement Learning](https://www.youtube.com/playlist?list=PL_iWQOsE6TfURIIhCrlt-wj9ByIVpbfGc) - UC Berkeley's course on deep RL
- [Advanced Deep Learning & Reinforcement Learning](https://www.youtube.com/playlist?list=PLqYmG7hTraZDNJre23vqCGIVpfZ_K2RZs) - DeepMind's advanced course

### Books

- [Algorithms for Reinforcement Learning](https://sites.ualberta.ca/~szepesva/papers/RLAlgsInMDPs.pdf) - Concise introduction to RL algorithms
- [Reinforcement Learning: Theory and Algorithms](https://rltheorybook.github.io/) - Theoretical foundations of RL
- [Deep Reinforcement Learning](https://www.manning.com/books/deep-reinforcement-learning) - Advanced RL with deep learning

### Projects

- [Gym](https://github.com/openai/gym) - OpenAI's toolkit for developing and comparing RL algorithms
- [RL Baselines3 Zoo](https://github.com/DLR-RM/rl-baselines3-zoo) - Collection of pre-trained RL agents
- [Highway-env](https://github.com/eleurent/highway-env) - Environment for autonomous driving tasks

## Advanced Resources

### Courses

- [CS285: Deep Reinforcement Learning](http://rail.eecs.berkeley.edu/deeprlcourse/) - Berkeley's advanced RL course
- [CS234: Reinforcement Learning](https://web.stanford.edu/class/cs234/) - Stanford's RL course
- [Deep RL Bootcamp](https://sites.google.com/view/deep-rl-bootcamp/lectures) - Intensive lectures on deep RL

### Books

- [Statistical Reinforcement Learning](https://www.routledge.com/Statistical-Reinforcement-Learning-Modern-Machine-Learning-Approaches/Sugiyama/p/book/9781439856895) - Modern machine learning approaches
- [Bandit Algorithms](https://tor-lattimore.com/downloads/book/book.pdf) - Comprehensive coverage of bandit algorithms
- [Reinforcement Learning and Optimal Control](http://web.mit.edu/dimitrib/www/RLbook.html) - Dimitri Bertsekas' advanced text

### Research

- [ICLR RL Papers](https://openreview.net/group?id=ICLR.cc) - RL papers from ICLR
- [arXiv Machine Learning](https://arxiv.org/list/cs.LG/recent) - Latest RL research papers
- [DeepMind Research](https://deepmind.com/research) - Cutting-edge RL research
